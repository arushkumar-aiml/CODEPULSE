# ⚡ CodePulse — Smart TODO Tracker for VS Code

CodePulse is a lightweight VS Code extension that helps developers track TODO, BUG, and FIXME comments across their entire project in one place.

---

## 🚀 Features

- 📝 Automatic detection of TODO, BUG, FIXME
- 📊 Grouped view with counts
- 🔄 Real-time updates on file save
- 🎯 One-click navigation to exact code line
- ⚡ Fast and lightweight

---

## 📸 How It Works

1. Open your project in VS Code  
2. CodePulse scans all files  
3. Displays grouped tasks in sidebar  
4. Click any item → jump to code  

---

## 🛠️ Tech Stack

- JavaScript  
- VS Code Extension API  
- Node.js
